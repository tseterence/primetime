About

PrimeTime is an interactive numbers game built to help users practice identifying prime numbers from composite numbers.


A prime number is a whole number greater than 1 that has exactly two factors - 1 and itself.



Rules

For each number shown, click "Yes" if the number is prime and "No" if it is not. Sort as many numbers correctly in the given time! But beware... one incorrect response and it's game over!
